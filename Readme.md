# SkillLink

SkillLink is a skill-sharing Android application that allows users to discover, share, and explore skills while connecting with mentors.

The application is built using Kotlin, Jetpack Compose, and Firebase.

## About the Project

SkillLink provides a platform where users can learn new skills and share their own knowledge with the community.

Users can create an account, manage their profile, publish skills, search for skills, save favorite skills, and contact mentors.

## Features

- User Registration and Login
- Firebase Authentication
- User Profile Management
- Edit Profile Information
- View Phone Number and Bio
- Publish New Skills
- Skill Category and Difficulty Selection
- Browse Available Skills
- Search Skills
- Filter Skills by Category
- View Skill Details
- Save Skills to Favorites
- Account-Based Persistent Favorites
- Contact Mentors through Email
- Dynamic Greeting on Home Screen
- Learning Progress Information
- Settings Screen
- Logout Functionality
- Firebase Firestore Integration

## Firebase Setup

This project uses Firebase Authentication and Cloud Firestore.

The `google-services.json` file is intentionally excluded from this repository for configuration and security purposes.

To run the project with your own Firebase project:

1. Create a Firebase project.
2. Register the Android application in Firebase.
3. Download your `google-services.json` file.
4. Place the file inside the `app/` directory.
5. Enable Firebase Authentication.
6. Enable Cloud Firestore.
